미디어 쿼리 : 출력장치 (모니터 태블릿 스마트폰)의 특징 가운데 일부를 참조해 css코드를 분기처리함으로써 하나의 html소스가 여러가지 뷰를 갖도록 구현할 수 있는 명세 (대부분 해상도에 따라 코드를 분기하는 법) @media only all and (조건문) { 실행코드 } <---- @media : 미디어 쿼리의 시작 선언 / 뒤에 오는 조건들과 사이에 공백이 있어야함 only : 미디어 쿼리를 지원하는 사용자만 미디어 쿼리 구문을 해석하라 (생략 가능) 일반적으로 작성하지 않음 all : 해석해야할 대상 미디어를 선언. all이 있다면 모든 미디어가 이 구문을 해석. -> all 대신에 screen print 같은 특정 미디어를 구체적으로 언급하기도 함. and : 논리적으로 and 연산을 수행해 앞 뒤의 조건을 모두 만족. or : 논리적으로 or 연산을 수행해 둘중 하나라도 만족하면 코드를 실행.
